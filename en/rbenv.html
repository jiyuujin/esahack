Ruby

Command Line Tools

Install Command Line Tools

xcode-select --install
xcodebuild -version

Set rbenv

Please install rbenv using Homebrew

brew install rbenv ruby-build

Set the config at .zshrc

sudo vi ~/.zshrc

export PATH="$HOME/.rbenv/bin:$PATH"
eval "$(rbenv init -)"

source ~/.zshrc

After setting, quit the terminal session

List of Ruby versions that can be installed

rbenv install --list
rbenv install 3.0.1
rbenv global 3.0.1
rbenv rehash
ruby -v

Install bundler

Confirm the bundler version

bundler -v

Install PostgreSQL

Install PostgreSQL

brew install postgresql

Rails setup

I couldn't install it properly without the following steps

gem install rake

New mimetic version licensed under MIT #41757

brew install shared-mime-info
update-mime-database /usr/local/share/mime

Install Rails, create the skeleton of your Rails app

gem install rails -v 5.2.3
rails _5.2.3_ new . -d postgresql --skip-bundle

Start at Rails

Install various gems

bundle install --path vendor/bundle

Start http://localhost:3000 with PostgreSQL

brew services start postgresql
rails s
